ILogos testing task version 1.0 22.03.2017

Программа которая с определенной периодичностью мониторит заданную директорию
на наличие новых файлов определенного XML-формата.
При обнаружении файла подходящего формата приложение сохраняет его содержимое в
аналогичную по структуре таблицу в PostgreSQL и перемещает файл в папку обработанных файлов,
а которые не удалось прочитать перемещает в другую папку.

Шаги для запуска программы:
1) Распаковать архив в удобное для Вас место на файловой системе;
2) Создайте базу данных в PostgreSQL;
3) Выполните 'table.sql' скрипт для создания необходимой по структуре тиблицы;
4) Используя файл 'config.properties' сконфигурируйте параметры подключения к базе данных и период мониторинга;
5) Из данной директории запустите коммандрую строку и выполните 'bin\xmlmonitor -c config.properties'.


<<<Формат поддерживаемого XML файла>>>

<Entry>
<!--строка длиной до 1024 символов-->
<content>Содержимое записи</content>
<!--дата создания записи-->
<creationDate>2014-01-01 00:00:00</creationDate>
</Entry>
